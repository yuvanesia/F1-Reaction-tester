# F1-Reaction-tester
F1 reaction time game (Lights Out start) built with HTML/CSS/JS. Tracks personal bests against a leaderboard. Built with AI assistance for design and code structure acceleration. Fully responsive.
The F1 Reflex Test is a web-based application designed to measure and track human reaction times using a gamified interface inspired by Formula 1 racing. The core objective of this project was to create an engaging, fast-loading tool that accurately simulates the "Lights Out" Grand Prix starting procedure. Built entirely using standard web technologies, the game provides an immediate, competitive benchmark against a high-score leaderboard while saving the user's personal best time locally.
